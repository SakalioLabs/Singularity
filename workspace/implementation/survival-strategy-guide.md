# Minecraft Survival Strategy Guide

## Phase 1: First 10 Minutes (Day 1)
1. Punch tree -> get 16 oak logs
2. Craft planks -> crafting table
3. Craft wooden pickaxe + wooden axe
4. Mine stone -> craft stone pickaxe + stone sword
5. Find coal or make charcoal (8 planks -> charcoal in furnace)
6. Craft torches (coal + stick)
7. Find or dig shelter location

## Phase 2: First Night
1. Build simple 5x5 shelter (dirt/wood/stone)
2. Place door + torches inside
3. Craft furnace if not done
4. Smelt any raw ore
5. Craft chest for storage
6. Wait for dawn or mine downward

## Phase 3: Resource Gathering (Day 2+)
1. Mine at y=16 for iron ore
2. Smelt iron -> iron ingots
3. Craft iron pickaxe + iron sword + iron armor
4. Find diamonds at y=-59
5. Build permanent base
6. Start farming (wheat, carrots)

## Phase 4: Progression
1. Enchanting table (4 obsidian + 2 diamond + 1 book)
2. Nether portal (10 obsidian)
3. Blaze rods from Nether fortress
4. Ender pearls from Endermen
5. Stronghold -> End portal

## Key Survival Rules
- Never mine straight down
- Always carry food
- Light up your base before night
- Keep backup tools
- Sleep in bed to set spawn point

# Minecraft Tech Tree

## Tier 0: Bare Hands
Punch trees -> oak_log, oak_planks, sticks, crafting_table

## Tier 1: Wood
wooden_pickaxe, wooden_axe, wooden_shovel, wooden_sword, furnace

## Tier 2: Stone
stone_pickaxe, stone_axe, stone_sword, torch, furnace

## Tier 3: Iron
iron_pickaxe, iron_axe, iron_sword, iron_armor (helmet/chest/legs/boots), shield, bucket, shears, crossbow

## Tier 4: Diamond
diamond_pickaxe, diamond_sword, diamond_armor, enchanting_table, jukebox

## Tier 5: Nether
nether_portal (10 obsidian), blaze_rod, netherite_scrap, golden_apple

## Tier 6: End
ender_pearl, end_portal, elytra, shulker_box, end_crystal

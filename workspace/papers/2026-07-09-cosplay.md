# Paper Card: COS-PLAY

**Title**: Co-Evolving LLM Decision and Skill Bank Agents for Long-Horizon Tasks
**Source**: https://arxiv.org/abs/2604.20987
**Date**: 2026
**Type**: Co-evolving game-agent skill bank and decision policy
**Core Problem**: Long-horizon game agents need to discover, retain, retrieve, and revise reusable skills across episodes instead of relying only on prompt history or static hand-authored skills.
**Core Method**: A decision agent retrieves skills from a learnable skill bank while a skill-bank agent segments unlabeled rollouts, learns compact effect contracts, curates skill quality, and updates the bank in a closed co-evolution loop.
**Memory / Task Mechanism**:
  - Keeps skill retrieval in the online decision loop, not only as offline documentation.
  - Represents reusable skills with protocols and effect contracts that can be matched against state and goals.
  - Uses rollout-derived evidence to refine, merge, split, or retire skill-bank entries over time.
**Borrowable Points**:
  - Retrieve skills by goal relevance, world-state fit, prerequisites, expected effects, and gate readiness.
  - Treat missing prerequisites and rejected gates as first-class readiness states instead of silently offering unusable skills.
  - Keep skill-bank maintenance auditable before any automatic runtime mutation is enabled.
**Singularity Adaptation**:
  - `skill-contract-report` now scores every known skill against a goal and world state, including required items, preconditions, postconditions, dependencies, and gate readiness.
  - `SkillLibrary.get_recommended_skills()` can surface contract-matched custom skills even before usage statistics accumulate, while skipping blocked skills.
  - Contract evidence complements `skill-graph-report`: the graph audits the bank structure, while the contract report audits retrieval readiness for a concrete Minecraft task.
**Next Action**: Run `skill-contract-report` on real approved skills and compare ready/review/blocked matches against planner choices on held-out crafting, mining, and building goals.

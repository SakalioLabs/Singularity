# Paper Card: AgentRx

**Title**: AgentRx: Diagnosing AI Agent Failures from Execution Trajectories

**Source**: https://arxiv.org/abs/2602.02475

**Date Reviewed**: 2026-07-10

**Core Method**: Normalizes heterogeneous agent traces, synthesizes global and trajectory-prefix constraints, evaluates guarded assertions into a step-indexed violation log, and uses the evidence log to identify the first unrecoverable failure plus one category from a cross-domain taxonomy.

**Evidence**:
- Releases 115 manually annotated failed trajectories spanning API workflows, incident management, and open-ended web/file tasks.
- Defines nine root-cause categories and separates all observed failures from the first failure that the run never recovers from.
- Reports 23.6 percentage points absolute improvement in failure localization and 22.9 points in root-cause attribution for its evaluated setup.
- Evaluates exact step accuracy, distance-tolerant accuracy, average step distance, and category accuracy rather than accepting a plausible explanation alone.

**Useful Transfer**:
- Treat Minecraft action schemas, verifier decisions, state effects, and prerequisites as executable trajectory constraints.
- Keep a compact, step-indexed violation log with supporting structured evidence.
- Mark a failed transition recovered only when a later observation proves the relevant state or action contract was repaired.
- Require manual critical-unit and category labels before trusting automated localization.

**Project Action**: `first_unrecovered_constraint_v1` evaluates deterministic Minecraft constraints over `minecraft_transition_unit_v1`, excludes repaired violations, and localizes the earliest high-confidence unresolved unit. `critical-transition-report` measures exact/within-one/category agreement and retains a recency baseline. It never grants runtime authority.

**Non-Claim**: Singularity has not implemented AgentRx's LLM constraint synthesis or adjudicator and has not reproduced its benchmark results. Current constraints are hand-specified, Minecraft-specific, and review-only.
